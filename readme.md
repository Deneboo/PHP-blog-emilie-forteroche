## Blog d'Emilie Forteroche

## Pour utiliser ce projet : 

- Commencer par cloner le projet. 

## Lancez le projet ! 

docker compose up --build

## installer autoload :

Dans la CMD du projet ou dans le bash (docker exec -it php-image-name bash) :
composer install

Puis :
composer dump-autoload
